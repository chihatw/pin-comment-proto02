# pin-comment-proto02

このプロジェクトは Next.js（App Router）+ TypeScript + Tailwind CSS v4 + shadcn/ui + pnpm で構築されています。

## 開発サーバーの起動

```sh
pnpm dev
```

## ビルド

```sh
pnpm build
```

## プロジェクトの特徴

- Next.js App Router 構成
- TypeScript
- Tailwind CSS v4
- shadcn/ui

## ホームページ

- `/` に「hello, world」と表示されます。
